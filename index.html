Tutorial Pasang Nyarch Linux

  1. Download ISO Nyarch Linux dari website resmi.
  2. Buat bootable USB pakai Rufus atau balenaEtcher.
  3. Boot dari USB lalu masuk ke live environment.
  4. Cek koneksi internet dengan ping archlinux.org.
  5. Atur partisi dengan fdisk atau cfdisk.
  6. Format partisi lalu mount ke /mnt.
  7. Install base system: pacstrap /mnt base linux linux-firmware.
  8. Generate fstab: genfstab -U /mnt >> /mnt/etc/fstab.
  9. Masuk ke sistem: arch-chroot /mnt.
  10. Atur zona waktu, locale, hostname, dan password root.
  11. Install bootloader (GRUB / systemd-boot).
  12. Reboot dan login ke sistem Nyarch Linux baru üéâ.
